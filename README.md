# 弹窗流demo

## 简介

> 项目使用第三方库`popperjs`进行tooltip定位，通过事件流的方式（类似lazyMan）进行弹窗控制，可实现新手弹窗功能。



## 运行

```
npm install
```

## 项目结构

```
─public
└─src
    ├─assets
    ├─common
    │  └─event
    ├─components
    │  ├─dialog
    │  ├─mask
    │  └─tooltip
    ├─pages
    │  ├─content
    │  ├─content1
    │  └─home
    ├─router
    └─store 
```
